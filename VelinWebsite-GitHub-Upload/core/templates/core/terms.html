Terms Of Service

One permanent link, managed responsibly.

Velin accounts are intended for legitimate professional identity sharing. Users are responsible for the content they publish and for programming the correct generated link onto each NFC card.

Account Responsibility

Each account owns one profile in this first version. Keep credentials secure and use a real email address that you can verify.

Publishing Rules

Profiles cannot be published until the email address is verified. The owner controls whether the profile is public or private.

NFC Link Handling

The generated public URL is intended to be the stable destination written to the card. Velin keeps that URL permanent so later edits do not break the tap experience.

Operational Changes

Velin may update pricing, product availability, or platform features over time while keeping shared profile links stable for active customers.
